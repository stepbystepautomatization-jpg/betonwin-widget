## Cómo completar este documento

1. **Lean cada pregunta** en la columna "Pregunta del cliente"
2. **Escriban la respuesta correcta** en la columna "Respuesta (completar)"
3. Si una pregunta no aplica, escriban "N/A" y el motivo
4. Si la respuesta depende del país, especifíquenlo (ej: "Chile: X, Argentina: Y")
5. Si no están seguros, escriban "POR VERIFICAR" y quién puede responder

**Una vez completado, estas respuestas se insertarán en la Knowledge Base del widget BetonBot para responder automáticamente a los clientes.**
